# 🎮 俄罗斯方块 (Tetris)

[![Version](https://img.shields.io/badge/version-1.4.0-blue.svg)](./)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](./)

一个使用 HTML5 Canvas + JavaScript 开发的纯静态前端俄罗斯方块游戏，支持桌面端和移动端，界面美观，操作流畅，像素级清晰渲染。

## ✨ 特性

- 🎯 **经典玩法**：完整的俄罗斯方块游戏规则，10列×20行标准尺寸
- 📱 **移动端适配**：支持手机触摸操作、横屏检测、响应式布局
- 🎨 **精美界面**：渐变色标题、毛玻璃效果、动画特效、圆角按钮
- ⚡ **流畅体验**：requestAnimationFrame 实现 60fps 游戏循环，触摸无延迟
- 🎮 **多种控制**：键盘、触摸滑动、方向按钮多种操作方式
- 🔊 **丰富音效**：7种不同音效（移动、旋转、下落、消除1-4行、升级、游戏结束）
- 🏆 **最高分记录**：自动保存最高分，新纪录特殊提示
- 📊 **等级系统**：最高 10 级，速度递增，难度提升
- ✨ **消除动画**：方块消除时带缩放、发光、闪光特效
- 🕹️ **长按连击**：左右方向键支持长按连续移动
- 🔄 **踢墙系统**：旋转时自动调整位置，支持靠墙旋转
- 💎 **像素级清晰**：优化渲染算法，方块边缘锐利无锯齿

## 🚀 快速开始

直接在浏览器中打开 `index.html` 文件即可开始游戏。

```bash
# 使用本地服务器（推荐）
npx serve .
# 或
python -m http.server 8080
```

然后在浏览器中访问 `http://localhost:8080`

## 🎮 操作说明

### 桌面端

| 按键 | 功能 |
|------|------|
| ← → 或 A D | 左右移动 |
| ↓ 或 S | 加速下落 |
| ↑ 或 W | 旋转方块 |
| 空格 | 暂停/继续 |
| 回车 | 开始游戏 |
| 🔊/🔇 按钮 | 开启/关闭音效 |

### 移动端

- **方向按钮**（横向排列）：
  - ↻ 旋转（蓝色高亮）
  - ← → 左右移动
  - ⤓ 直接掉落（橙色高亮）
  - 🔊/🔇 音效开关
- **触摸滑动**：在画布上滑动控制方向
- **长按连击**：按住左右按钮可连续移动
- **开始按钮**：底部独立宽按钮
- **横屏检测**：横屏时自动暂停并提示竖屏游玩

## 🎯 游戏规则

1. **方块形状**：7 种经典方块（I、O、T、S、Z、J、L）
2. **消除机制**：填满一行即可消除，多行同时消除得分更高
3. **等级系统**：
   - 初始等级 1，每消除 10 行升 1 级
   - 最高 10 级，等级越高下落速度越快
   - 1 级：1000ms，10 级：100ms
4. **得分规则**：
   - 消除 1 行：100 × 等级
   - 消除 2 行：300 × 等级
   - 消除 3 行：600 × 等级
   - 消除 4 行：1000 × 等级
   - 软降（↓键）：每行 1 分
   - 硬降（⤓按钮）：每行 2 分
5. **踢墙系统**：
   - 旋转时如果碰到墙壁或其他方块，会自动尝试左右移动来适应旋转
   - 支持左右偏移和向上偏移（针对I方块）
6. **阴影提示**：
   - 1 级：30% 透明度
   - 2 级：15% 透明度
   - 3 级及以上：无阴影（增加难度）

## 🔧 技术实现

### 核心类 `TetrisGame`

```javascript
class TetrisGame {
    // 游戏配置
    cols = 10;        // 列数（标准10列）
    rows = 20;        // 行数（标准20行）
    blockSize = 25;   // 方块大小（像素）
    
    // 游戏状态
    board[][];        // 游戏板二维数组
    currentPiece;     // 当前方块
    nextPiece;        // 下一个方块
    score;            // 得分
    highScore;        // 最高分（本地存储）
    level;            // 等级（1-10）
    lines;            // 已消除行数
    
    // 消除动画
    clearingLines[];  // 正在消除的行
    clearAnimationProgress;  // 动画进度 0-1
    
    // 音效管理
    audio;            // AudioManager 实例
    
    // 移动端优化
    setupMobileButton();  // 无延迟触摸+长按连击
}
```

### 关键方法

- `start()` - 开始游戏
- `rotate()` - 旋转方块（矩阵转置算法）
- `move(dir)` - 左右移动
- `drop()` - 下落一格
- `hardDrop()` - 直接掉落到底部
- `clearLines()` - 检测并消除完整行
- `draw()` - 绘制游戏画面
- `drawClearingBlock()` - 绘制消除动画
- `drawShadow()` - 绘制落点阴影
- `setupMobileButton(btnId, action)` - 移动端按钮优化（无延迟+长按）
- `loadHighScore()` - 从本地存储加载最高分
- `saveHighScore(score)` - 保存最高分到本地存储

### 方块定义

```javascript
pieces = {
    I: { shape: [[1,1,1,1]], color: '#00f0f0' },           // 长条 - 青色
    O: { shape: [[1,1],[1,1]], color: '#f0f000' },          // 正方形 - 黄色
    T: { shape: [[0,1,0],[1,1,1]], color: '#a000f0' },     // T形 - 紫色
    S: { shape: [[0,1,1],[1,1,0]], color: '#00f000' },     // S形 - 绿色
    Z: { shape: [[1,1,0],[0,1,1]], color: '#f00000' },     // Z形 - 红色
    J: { shape: [[1,0,0],[1,1,1]], color: '#0000f0' },     // J形 - 蓝色
    L: { shape: [[0,0,1],[1,1,1]], color: '#f0a000' }      // L形 - 橙色
}
```

### 音效系统

使用 Web Audio API 生成程序化音效，无需外部音频文件：

```javascript
// 音效类型
- playMove()      // 移动音效
- playRotate()    // 旋转音效  
- playDrop()      // 下落音效
- playClear(n)    // 消除音效（1-4行递进）
- playLevelUp()   // 升级音效
- playGameOver()  // 游戏结束音效
```

### 最高分记录

使用 localStorage 持久化存储最高分：

```javascript
// 自动保存
if (this.score > this.highScore) {
    this.highScore = this.score;
    localStorage.setItem('tetrisHighScore', this.highScore);
}

// 新纪录提示
游戏结束界面显示 "🏆 新纪录!" 动画
```

### 移动端触摸优化

```javascript
// 1. 消除300ms点击延迟
btn.addEventListener('touchstart', (e) => {
    e.preventDefault();  // 阻止默认行为
    doAction();          // 立即执行
}, {passive: false});

// 2. 长按连续移动
longPressTimer = setTimeout(() => {
    repeatTimer = setInterval(() => {
        doAction();      // 每80ms执行一次
    }, 80);
}, 200);  // 长按200ms后开始连击

// 3. 触觉反馈
if (navigator.vibrate) {
    navigator.vibrate(10);  // 10ms轻微震动
}
```

## 📱 响应式设计

### 断点设置

- **桌面端**：> 900px，显示三栏布局，固定画布尺寸
- **移动端**：≤ 900px，显示单栏布局，自适应画布宽度
- **小屏幕**：≤ 480px，紧凑布局，缩小按钮尺寸

### 移动端优化

- **画布自适应**：`width: 92vw, max-width: 340px, max-height: 60vh`
- **禁用双击缩放**：`touch-action: manipulation`
- **去除点击高亮**：`-webkit-tap-highlight-color: transparent`
- **触摸无延迟**：使用 `touchstart` 代替 `click`
- **长按连击**：左右按钮支持长按连续移动
- **横屏检测**：自动检测并提示竖屏游玩

### 视觉设计

- **信息栏**：宽度与操作按钮对齐，圆角外框
- **操作按钮**：圆形设计，52-56px，渐变+阴影立体感
- **旋转按钮**：蓝色渐变高亮
- **掉落按钮**：橙色渐变高亮
- **开始按钮**：胶囊形设计，宽度85%
