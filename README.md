# 🎮 俄罗斯方块 (Tetris)

一个使用 HTML5 Canvas + JavaScript 开发的纯静态前端俄罗斯方块游戏，支持桌面端和移动端，界面美观，操作流畅。

## ✨ 特性

- 🎯 **经典玩法**：完整的俄罗斯方块游戏规则
- 📱 **移动端适配**：支持手机触摸操作和横屏检测
- 🎨 **精美界面**：渐变色标题、毛玻璃效果、动画特效
- ⚡ **流畅体验**：requestAnimationFrame 实现 60fps 游戏循环
- 🎮 **多种控制**：键盘、按钮、触摸滑动多种操作方式
- 📊 **等级系统**：最高 10 级，速度递增，难度提升
- ✨ **消除动画**：方块消除时带缩放、发光、闪光特效

## 🚀 快速开始

直接在浏览器中打开 `index.html` 文件即可开始游戏。

```bash
# 使用本地服务器（推荐）
npx serve .
# 或
python -m http.server 8080
```

然后在浏览器中访问 `http://localhost:8080`

## 🎮 操作说明

### 桌面端

| 按键 | 功能 |
|------|------|
| ← → 或 A D | 左右移动 |
| ↓ 或 S | 加速下落 |
| ↑ 或 W | 旋转方块 |
| 空格 | 暂停/继续 |
| 回车 | 开始游戏 |

### 移动端

- **圆形控制盘**：
  - ↻ 旋转
  - ← → 左右移动
  - ↓ 加速下落（支持长按快速下落）
- **触摸滑动**：在画布上滑动控制方向
- **横屏检测**：横屏时自动暂停并提示竖屏游玩

## 🎯 游戏规则

1. **方块形状**：7 种经典方块（I、O、T、S、Z、J、L）
2. **消除机制**：填满一行即可消除，多行同时消除得分更高
3. **等级系统**：
   - 初始等级 1，每消除 10 行升 1 级
   - 最高 10 级，等级越高下落速度越快
   - 1 级：1000ms，10 级：100ms
4. **得分规则**：
   - 消除 1 行：100 × 等级
   - 消除 2 行：300 × 等级
   - 消除 3 行：600 × 等级
   - 消除 4 行：1000 × 等级
5. **阴影提示**：
   - 1 级：30% 透明度
   - 2 级：15% 透明度
   - 3 级及以上：无阴影（增加难度）

## 🔧 技术实现

### 核心类 `TetrisGame`

```javascript
class TetrisGame {
    // 游戏配置
    cols = 15;        // 列数
    rows = 25;        // 行数
    blockSize = 20;   // 方块大小
    
    // 游戏状态
    board[][];        // 游戏板二维数组
    currentPiece;     // 当前方块
    nextPiece;        // 下一个方块
    score;            // 得分
    level;            // 等级（1-10）
    lines;            // 已消除行数
    
    // 消除动画
    clearingLines[];  // 正在消除的行
    clearAnimationProgress;  // 动画进度 0-1
}
```

### 关键方法

- `start()` - 开始游戏
- `rotate()` - 旋转方块
- `move(dir)` - 左右移动
- `drop()` - 下落一格
- `clearLines()` - 检测并消除完整行
- `draw()` - 绘制游戏画面
- `drawClearingBlock()` - 绘制消除动画
- `drawShadow()` - 绘制落点阴影
